# 사고 모드 (Thinking)

사고 모드는 AI 모델이 응답 전에 수행하는 추론의 깊이를 제어합니다. 높은 레벨일수록 더 깊이 생각하지만 토큰 소비와 응답 시간이 증가합니다.

## 사고 레벨

| 레벨      | 별칭           | 토큰 예산 | 설명             |
| --------- | -------------- | --------- | ---------------- |
| `off`     | —              | 0         | 사고 없음        |
| `minimal` | "think"        | 1024      | 최소 사고        |
| `low`     | "think hard"   | 4096      | 낮은 사고        |
| `medium`  | "think harder" | 10240     | 중간 사고 (기본) |
| `high`    | "ultrathink"   | 32768     | 높은 사고        |
| `xhigh`   | "ultrathink+"  | 65536     | 최고 사고        |

### 모델별 지원

| 모델                    | 지원 레벨                         |
| ----------------------- | --------------------------------- |
| Anthropic Claude        | 모든 레벨                         |
| OpenAI GPT-5.2 / Codex | `xhigh`만 지원                    |
| xAI Grok                | 바이너리 (on/off)                 |
| 기타                    | 모델에 따라 다름                  |

## 디렉티브 문법

세 가지 형식을 지원합니다:

```
/think <level>
/t <level>
/think:<level>
/thinking <level>
```

### 메시지와 함께 사용

```
/think high 이 알고리즘의 시간 복잡도를 분석해줘
```

에이전트는 `high` 레벨로 사고한 후 "이 알고리즘의 시간 복잡도를 분석해줘"를 처리합니다.

### 세션 기본값 설정

디렉티브만 단독으로 보내면 세션 기본값으로 설정됩니다:

```
/think medium
```

이후 모든 메시지는 `medium` 레벨로 처리됩니다.

## 해석 우선순위

1. **인라인 디렉티브** — 해당 메시지에만 적용
2. **세션 오버라이드** — 단독 디렉티브로 설정
3. **전역 기본값** — 설정 파일
4. **폴백** — 추론 가능 모델은 `low`, 아니면 `off`

## 설정

### 전역 기본값

```json5
{
  agents: {
    defaults: {
      thinking: "medium",
    },
  },
}
```

### 에이전트별

```json5
{
  agents: {
    list: [
      {
        id: "deep-thinker",
        thinking: "high",
      },
      {
        id: "quick-responder",
        thinking: "off",
      },
    ],
  },
}
```

## 상세 모드 (Verbose)

도구 실행 결과를 채팅에 표시합니다:

```
/verbose on           # 도구 메타데이터 전송
/verbose full         # 메타데이터 + 도구 출력 전송
/verbose off          # 기본값
/v on                 # 축약형
```

- **`on`**: 도구 실행 결과를 구조화된 메타데이터로 별도 메시지 전송
- **`full`**: 메타데이터에 더해 도구 출력도 전달
- **`off`**: 에이전트 응답만 전달 (기본)

## 추론 표시 (Reasoning)

에이전트의 추론 과정을 별도 메시지로 표시합니다:

```
/reasoning on         # 추론 과정 표시
/reasoning off        # 숨기기 (기본)
/reasoning stream     # 스트리밍으로 표시
```

활성화하면 "Reasoning:" 접두사가 붙은 별도 메시지가 응답 전에 전송됩니다.

## 비용 고려사항

| 레벨   | 토큰 예산 | 비용 영향      | 권장 용도              |
| ------ | --------- | -------------- | ---------------------- |
| `off`  | 0         | 최소           | 간단한 질문            |
| `low`  | 4096      | 낮음           | 일반 대화              |
| `medium` | 10240   | 중간           | 코딩, 분석 (기본 추천) |
| `high` | 32768     | 높음           | 복잡한 문제 해결       |
| `xhigh`| 65536     | 매우 높음      | 최대 성능 필요 시      |
